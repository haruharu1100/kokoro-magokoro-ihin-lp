# 公式LINE 自動返信 作成ガイド

## やりたいこと

LPで入力された下記内容を、LINE送信時の本文に反映する。

- 名前
- 電話番号
- エリア
- 相談内容
- 詳細

その後、LINE側で内容に応じて自動返信する。

## サイト側の送信文

LPからLINEへ送る本文は下記形式。

```text
無料見積もりをお願いします。

お名前: 山田 太郎
電話番号: 090-0000-0000
エリア: 大阪市北区
相談内容: 遺品整理
詳細: 1LDK、荷物多め、今月中希望、立ち会い可能です。

間取り、物量、希望日、立ち会い可否が分かれば追って送ります。
```

## 公式LINEの最初のあいさつ

```text
お問い合わせありがとうございます。
心まごころ遺品整理です。

遺品整理、生前整理、供養、買取のご相談を承っております。
大阪、兵庫、奈良、京都エリアで対応可能です。

概算見積もりのため、分かる範囲で下記を教えてください。

1. 作業場所の市区町村
2. 間取り
3. 物量
4. 希望時期
5. 立ち会いの可否
6. 写真があれば数枚

急ぎの場合はお電話でも対応できます。
0745-51-1665

営業時間は9:00〜18:00です。
```

## 内容がある程度そろっている場合

```text
ありがとうございます。
ご入力内容を確認しました。

担当者が内容を確認して返信いたしますので、少々お待ちください。

お部屋全体、押し入れ、キッチン、玄関まわり、家具家電の量が分かる写真があれば、続けて送っていただくと概算見積もりがスムーズです。
```

## 情報が足りない場合

```text
ありがとうございます。
概算見積もりのため、追加で分かる範囲を教えてください。

1. 間取り
2. 物量
3. 希望時期
4. 立ち会いの可否
5. 写真の送付可否

分かる範囲だけで大丈夫です。
```

## クレーム・トラブル時

```text
ご不安な思いをさせてしまい申し訳ございません。
内容を正確に確認し、担当者より直接ご連絡いたします。

恐れ入りますが、お名前、お電話番号、状況の詳細をお送りください。
```

## AIに任せる範囲

- 最初のあいさつ
- 不足情報の聞き返し
- 写真送付の案内
- 担当者返信待ちの案内
- クレーム時の一次受け

## 人に引き継ぐ条件

- クレーム
- 金額の確定
- 日程の確定
- 急ぎ案件
- 特殊清掃が必要そうな相談
- 法的、相続、賃貸退去トラブルが絡む相談

## 実装時に必要な情報

秘密情報はチャットに貼らず、環境変数に入れる。

```text
LINE_CHANNEL_SECRET=
LINE_CHANNEL_ACCESS_TOKEN=
```

## LINE公式アカウント側で必要な設定

- Messaging APIを有効にする。
- Webhook URLを設定する。Vercel公開後は `https://公開URL/api/line-webhook` を入れる。
- Use webhookをONにする。
- AI返信と二重返信になる場合、通常の自動応答はOFFにする。
- あいさつメッセージだけ残すか、AI側に統一するかを決める。
