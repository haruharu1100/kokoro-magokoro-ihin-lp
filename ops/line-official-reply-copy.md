# 公式LINE側に入れる文言

## 推奨設定

AI生成は使わず、Webhookのテンプレ自動返信で運用する。
LINE公式側の応答メッセージは二重返信防止のためOFFにする。

```text
チャット: オン
Webhook: オン
応答メッセージ: オフ
応答方法: 手動チャット
あいさつメッセージ: オン
```

## あいさつメッセージ

友だち追加時だけ送る文言。

```text
友だち追加ありがとうございます。
心まごころ遺品整理です。

遺品整理・生前整理・供養・買取のご相談を承っております。
大阪、兵庫、奈良、京都エリアで対応可能です。

概算見積もりをご希望の場合は、分かる範囲で下記をお送りください。

1. 作業場所の市区町村
2. 間取り
3. 物量
4. 希望時期
5. 立ち会いの可否
6. お名前・お電話番号
7. 写真があれば数枚

写真を送っていただくと、概算見積もりがスムーズです。
営業時間は9:00〜18:00です。
お急ぎの場合は 0745-51-1665 へお電話ください。
```

## 応答メッセージ

Webhookが返信するため、LINE公式の応答メッセージは使わない。
もし一時的にWebhookをOFFにする場合だけ、下記を使う。

```text
お問い合わせありがとうございます。
心まごころ遺品整理です。

内容を確認し、担当者より返信いたします。
概算見積もりをご希望の場合は、作業場所、間取り、物量、希望時期、立ち会い可否、写真をお送りください。
```

## テンプレ自動返信の分岐

- テスト: テスト受信の案内
- 情報なし: 見積もりに必要な項目を案内
- 見積情報あり: 名前と電話番号の送付を依頼
- 名前・電話あり: 詳細情報があれば担当者確認、なければ追加項目を案内
- 見積情報 + 名前・電話あり: 担当者確認待ちを案内
- クレーム・特殊清掃・トラブル: 担当者引き継ぎを案内
